═══════════════════════════════════════════
TROUBLESHOOTING
═══════════════════════════════════════════

If "DATABASE_URL not set": Re-create the PostgreSQL database in Replit Database tab.

If "ANTHROPIC_API_KEY missing": Backend will start but content generation will fail. User must add this secret.

If frontend shows blank/error: Check browser console + look for "Module not found" — usually means cd frontend && npm install didn't complete.

If port already in use:
lsof -ti:3000 | xargs kill -9 2>/dev/null
lsof -ti:3001 | xargs kill -9 2>/dev/null

If schema.sql fails with "relation already exists": That's fine — the schema starts with DROP TABLE IF EXISTS, so it's idempotent. Re-run it.
